# 쨖쨔ㄲ (Clap Rythm)

'쨖쨔ㄲ'은 YouTube 동영상을 보면서 박수를 치는 리듬게임입니다.<br>
리듬게임도 즐기고 여러분의 건강도 증진시킬 수 있습니다!<br>
<br>
'쨖쨔ㄲ'은 웹 프레임워크 Flask로 만들었으며 Google App Engine 위에서 동작합니다.<br>
주소 https://clap.heek.kr<br>
